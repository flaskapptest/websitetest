Welcome to my website!

This site was created using the Flask framework on top of Python 3, and hosted on PythonAnywhere

  • Python

    • Python is a simple high level interpreted programming language designed for ease of use.

  • Flask

    • Flask is a python framework for developing web applications.

  • PythonAnywhere

    • Python Anywhere is a gratis service to host, run, and code Python on someone else's servers.

    Why did I choose to host this site on Python Anywhere?

    I wanted to use my Raspberry Pi as a server for this site and use DuckDNS for my domain name, but didn't want to open up my LAN up to the world, and really didn't want deal with the constant port scans that accompany that.

    This site seems to be very simplistic?!

    Yes!! This is by design, too many sites these days go overboard with the JavaScript, which is really lousy for users with a slow internet connection or users who use text only browsers. This site is almost entirely HTML, with a little bit of bootstrap for the CSS. I leave no cookies (although bootstrap and Python Anywhere both leave one), no JavaScript, and few images, for a snappy performance.
